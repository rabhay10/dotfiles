**RedOrchid** is a custom Ranger file manager colorscheme designed for enhanced readability, clear visual hierarchy, and aesthetic appeal. Its carefully chosen red and pink palette highlights directories, files, executables, symlinks, images, and selections, making navigation intuitive and visually striking.

## Features

- Bold, soft red/pink tones for main text (`rose`)  
- Deep red for directories (`crimson`)  
- Strong red highlights for executables and marked files (`accent`)  
- Orange accents for symlinks and media files (`orchid`)  
- Clear visual cues for selections, errors, and urgent items  
- Minimalistic and distraction-free design  

## Installation

```bash
mkdir -p ~/.config/ranger/colorschemes
ln -s ~/dotfiles/ranger/colorschemes/redorchid.py ~/.config/ranger/colorschemes/redorchid.py
:colorscheme redorchid
